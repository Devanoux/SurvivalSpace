## Installation
#### 1. Clone this repository in your machine

  ```bash
  git clone git@github.com:Devanoux/42_FtShmup.git SurvivalSpace
  cd SurivalSpace
  ```
#### 2. Install dependencies  
 You need c compiler `cc`  
 You need `ncurses` library  

## Compilation

To build the project, use the provided `Makefile`:

```bash
 make
```

To clean up the project, remove binaries and object files:

```bash
make fclean
```

## Launch

To launch the game, execute the binary  

```bash
./SurvivalSpace
```
